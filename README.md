# What you can do
Build the executable file on your local.
```
task build
```

Run the cli command.
```
./bin/go-cli get [gopherName]

Ex) ./bin/go-cli get 5th-element
```

Clean all png files.
```
task clean
```


## about get command
This get command will calll GitHub repository in order to return the desired Gopher.
```
Usage:
  go-cli get [flags]

Flags:
  -h, --help   help for get
```
